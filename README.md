# flarum-confix-theme
flarum-confix-theme

### 演示

[点击查看](https://flarum.pro/)

### 安装

使用 composer 命令进行安装:

```sh
composer require yannisme/confixtheme
```

### 更新

```sh
composer update yannisme/confixtheme
php flarum cache:clear
```
